
Это программа выводит информацию в заданном виде:
Код: 84975, Артикул: 10967, Наименование: Преобразователь видеосигнала BNC в HDMI/VGA/CVBS AD001AHD, 
Производитель: SC&T, Единица измерения: шт, Розничная цена: 7075.00р.

Если нет значения у полей "Артикул" и "Производитель" в файле .xlsx, то в .txt этим полям присваивается значение "Нет".

Настроен ввод path для обоих файлов, но можно указать в переменных pathXls и pathTxt, соответственно 
с закомментированием ненужных переменных pathXls и pathTxt.
